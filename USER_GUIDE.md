## 1. Установка и обновление

### Установка

1. Скачайте установщик `.exe`.
2. Запустите — пройдите стандартный мастер установки.
3. Приложение появится в системном трее (иконка рядом с часами).
4. Левый клик по иконке трея — открыть/скрыть доску.

### Где хранятся данные

Все данные хранятся в папке:

```
C:\Users\<Имя пользователя>\AppData\Roaming\task-grabber\
```

- `tasks.db` — база данных со всеми задачами, колонками, заметками.
- `backups\` — автоматические и ручные бэкапы.
- `attachments\` — прикреплённые файлы.

### Обновление

При обновлении до новой версии данные **сохраняются**. Установщик заменяет только файлы приложения, не трогая папку с данными в AppData.

## 2. Быстрый старт

### Первый запуск

При первом запуске автоматически создаются 5 колонок:

| Колонка | Назначение |
|---------|------------|
| Новые | Все новые задачи попадают сюда |
| В работе | То, над чем работаете прямо сейчас |
| Ждём | Заблокировано, ждём ответа/события |
| Готово | Выполненные задачи |
| Забито | Отменённые или неактуальные |

### Основные понятия

- **Задача** — единица работы с заголовком, описанием, приоритетом, тегами и вложениями.
- **Колонка** — этап работы (можно добавлять свои).
- **Доска** — независимый набор колонок и задач (например: "Работа", "Личное").
- **Источник** — откуда взялась задача: вручную, из текста, из файла, из письма.
- **Приоритет** — нет / низкий / средний / высокий (цветная полоска на карточке).

## 3. Интерфейс

```
┌─────────────────────────────────────────────────────────────────┐
│ ◉ Task Grabber  [Доска ▾]    🔍 Поиск...   [+ Задача]  [AI] [⚙] [—×]│
├────────┬────────────────────────────────────────────────────────┤
│        │                                                        │
│ Поиск  │  📥 Новые    🔄 В работе   ⏸ Ждём   ✅ Готово  💀    │
│ #теги  │  ┌────────┐  ┌────────┐                               │
│ Приор. │  │ Task 1 │  │ Task 3 │                               │
│ Источн.│  │ 📎 2   │  │ #тег   │                               │
│ Заметки│  └────────┘  └────────┘                               │
│ Архив  │                                                        │
│ Стата  │  ┌ ─ ─ ─ Drop zone: файлы и .msg ─ ─ ─ ─ ─ ─ ─ ─ ┐  │
│        │  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘  │
├────────┴────────────────────────────────────────────────────────┤
│ Задач: 12  │  Сегодня: 3  │  Ctrl+Shift+T — захват текста      │
└─────────────────────────────────────────────────────────────────┘
```

### Title Bar (верхняя панель)

- **Логотип** — перетаскивание окна.
- **[Доска ▾]** — переключатель досок (BoardSwitcher).
- **[+ Задача]** — создать задачу вручную.
- **[AI]** — открыть AI помощника.
- **[⚙]** — настройки.
- **[— □ ×]** — свернуть, развернуть, закрыть.

### Sidebar (боковая панель)

Можно свернуть кнопкой `←` в заголовке. Содержит:

- **Поиск** — fuzzy-поиск по заголовкам и описаниям.
- **Фильтры** — по тегам, приоритету, источнику задачи.
- **Заметки** — быстрые заметки (не задачи).
- **Архив** — просмотр заархивированных задач.
- **Статистика** — графики по задачам.

### Status Bar (нижняя строка)

Показывает количество задач, сколько создано сегодня и напоминание о хоткеях.

### Карточка задачи

- Цветная полоска слева — приоритет (синий/жёлтый/красный).
- Иконка источника — ручная ✋, текст 📝, файл 📁, письмо ✉️.
- **📎 N** — количество вложений.
- Цветные бейджи — теги.
- Дата — относительная ("2ч назад").

## 4. Создание задач

### Кнопка "+ Задача"

Нажмите кнопку в title bar — откроется диалог. Заполните заголовок, опционально — описание, колонку, приоритет, теги, дедлайн, вложения.

### Ctrl+Shift+T — из выделенного текста

1. Выделите текст в любом приложении (браузер, Word, почта, etc.).
2. Нажмите `Ctrl+Shift+T`.
3. Task Grabber откроется с диалогом — текст уже вставлен в описание, первая строка — в заголовок.
4. Отредактируйте и нажмите **Создать**.

### Instant Capture — двойное быстрое нажатие

Нажмите `Ctrl+Shift+T` дважды быстро (менее 500мс между нажатиями) — задача создастся **мгновенно** без диалога. Заголовок = первая строка буфера обмена. Появится toast-уведомление с названием задачи — можно кликнуть, чтобы открыть её.

> Пример: скопировали ссылку, быстро два раза нажали — задача создана.

### Ctrl+Shift+F — из файлов

1. Выберите файлы в Проводнике.
2. Нажмите `Ctrl+Shift+F`.
3. Откроется диалог создания задачи, файлы добавятся как вложения.

### Ctrl+Shift+S — скриншот-задача

Нажмите `Ctrl+Shift+S` — откроется диалог с заголовком-датой. Прикрепите скриншот вручную через поле вложений.

### Drag & Drop

Перетащите прямо на доску:

| Что | Результат |
|-----|-----------|
| Файлы (любые) | Задача с именем файла + файл как вложение |
| `.msg` файл (Outlook) | Задача с темой письма + тело письма в описании |
| Текст из браузера | Задача с выделенным текстом |

Drop zone — пунктирная область внизу доски. Также работает перетаскивание на **виджет**.

### System Tray меню

Правый клик на иконке в трее:

- **Новая задача** (`Ctrl+Shift+T`) — открыть диалог.
- **Задача из файлов** (`Ctrl+Shift+F`) — открыть диалог.
- **Открыть доску** — показать главное окно.
- **Настройки** — открыть настройки.
- **Выход** — закрыть приложение.

### Toast-уведомления

После создания задачи через Instant Capture или захват файлов появляется всплывающее уведомление в углу экрана. Клик по нему открывает созданную задачу.

## 5. Управление задачами

### Перемещение между колонками

Перетащите карточку мышью из одной колонки в другую. Поддерживаются плавные анимации.

### Открыть задачу

Кликните по карточке — откроется детальный вид.

### В детальном виде (модалка)

- **Редактировать заголовок** — клик на заголовок.
- **Описание** — поддерживает Markdown. Переключайтесь между режимом редактирования и просмотра кнопкой 👁/✏️.
- **Чеклисты** — пишите в описании в формате Markdown: `- [ ] пункт` / `- [x] готово`. Чекбоксы кликабельны прямо в просмотре.
- **Приоритет** — выбирайте из выпадающего списка (none/низкий/средний/высокий).
- **Дедлайн** — date picker, опционально. Задачи с просроченным дедлайном подсвечиваются.
- **Напоминание** — конкретная дата/время, придёт системное уведомление.
- **Теги** — начните вводить тег в поле, выберите из autocomplete или создайте новый. У каждого тега свой цвет.
- **Колонка** — переместить задачу в другую колонку без drag&drop.
- **Вложения** — кнопка "Прикрепить файл", просмотр прикреплённых файлов. Картинки показывают превью при наведении.
- **Конфиденциально** — пометить задачу как приватную (AI не будет отправлять её содержимое в открытом виде).
- **Архивировать** — скрыть задачу с доски, сохранив в архиве.
- **Удалить** — безвозвратное удаление.
- **Focus Mode** — открыть задачу в режиме фокуса.
- **Связанные задачи** — привязать другие задачи.

## 6. Множественные доски

Несколько независимых досок — для разных контекстов (например: "Работа", "Личное", "Проект X").

### Переключение досок

В title bar есть кнопка **[Название доски ▾]** — кликните, чтобы открыть список досок. Выберите нужную — содержимое доски сразу переключится. Также можно переключиться через Global Search (`Ctrl+Space`).

### Создание доски

1. Нажмите **[Название доски ▾]** → **+ Новая доска**.
2. Введите название.
3. Выберите цвет-метку из палитры.
4. Нажмите **Создать** — доска станет активной.

### Переименование доски

В выпадающем списке наведите на доску → нажмите иконку карандаша → введите новое название → Enter.

### Удаление доски

В выпадающем списке наведите на доску → нажмите иконку корзины → подтвердите. Нельзя удалить последнюю оставшуюся доску.

> Каждая доска имеет свои колонки. При переключении доски фильтры и поиск сбрасываются.

## 7. Виды отображения

Доску можно смотреть в трёх режимах. Переключатель находится в панели над колонками.

### Kanban (по умолчанию)

Классическая колоночная доска с карточками задач. Drag & drop между колонками.

### Timeline (Шкала времени)

Горизонтальная шкала с задачами, у которых указан дедлайн. Задачи отображаются как полосы на временной оси.

- **Навигация** — кнопки `<` и `>` для перемотки недель.
- **Drag & drop** — перетаскивайте полосу задачи, чтобы изменить её даты прямо на шкале.
- **Клик по задаче** — открывает детальный вид.
- Задачи без дедлайна в Timeline не отображаются.

### Calendar (Календарный вид)

Месячная сетка. Каждая ячейка — один день, внутри показаны задачи с дедлайном на этот день.

- **Навигация** — кнопки `<` и `>` для переключения месяца.
- **Drag & drop** — перетащите карточку задачи на другой день, чтобы изменить дедлайн.
- **Клик по задаче** — открывает детальный вид.
- Сегодняшний день подсвечивается.

## 8. Массовые действия

Позволяют выбрать несколько задач и выполнить действие сразу над всеми.

### Выбор задач

- **Shift+клик** — выбрать диапазон задач в колонке.
- **Ctrl+клик** — добавить/убрать задачу из выделения.

### Batch Toolbar

Когда выбрана хотя бы одна задача, внизу экрана появляется панель массовых действий:

```
┌──────────────────────────────────────────────────────────┐
│ ☑ 3 задачи  │  Переместить ▾  │  Архивировать  │  Удалить  │  ✕ │
└──────────────────────────────────────────────────────────┘
```

- **Переместить** — выбрать колонку из выпадающего меню, все выбранные задачи переедут туда.
- **Архивировать** — заархивировать все выбранные задачи.
- **Удалить** — удалить (с подтверждением).
- **✕** — сбросить выделение.

## 9. Focus Mode

Focus Mode — отдельное окно для работы над одной задачей с таймером Помодоро.

### Запуск

- Кнопка **Timer** (иконка таймера) в детальном виде задачи.
- Хоткей `Ctrl+Shift+F2`.
- Command Palette → "Focus Mode".

### Интерфейс

```
┌──────────────────────┐
│ ≡  Focus Mode   — ×  │
├──────────────────────┤
│ [Выбрать задачу...  ▾]│
│                      │
│      25:00           │ ← таймер (кольцо)
│      Фокус           │
│                      │
│   [↺]  [▶]  [✕]     │ ← сброс / пауза-пуск / стоп
│   3м    итого: 27м   │
│                      │
│   [✅ Готово — завершить задачу]  │
├──────────────────────┤
│ ЧЕКЛИСТ          0/3 │
│ ☐ Написать тесты     │
│ ☑ Поправить баг      │
│ [Добавить пункт...]+  │
└──────────────────────┘
```

### Pomodoro

- **25 минут** — рабочий блок (Фокус).
- **5 минут** — перерыв (автостартует после рабочего блока).
- По завершении каждого блока приходит системное уведомление.
- Счётчик помидоров отображается в заголовке окна (🍅).

### Time Tracking

Пока таймер идёт, приложение считает потраченное время. Счётчик сессии и общее время задачи видны под таймером. Данные сохраняются в базе.

### Чеклист

- Пункты из Markdown-описания задачи (`- [ ] ...`) автоматически подтягиваются.
- Можно добавлять локальные пункты прямо в Focus Mode.
- Отметка чекбокса синхронизируется с описанием задачи.

### Мини-режим

Кнопка свернуть (`—`) сворачивает окно в компактный виджет-таймер. Показывает только время и кнопку пауза/старт. Развернуть обратно — кнопка `□`.

## 10. Quick Notes

Быстрые заметки — для мыслей, которые не обязательно оформлять как задачу.

### Создание

- Хоткей `Ctrl+Shift+N` — появляется маленький диалог "Быстрая заметка".
- Напечатайте текст, нажмите **Enter** или кнопку "Сохранить" — заметка сохранена.

### Просмотр

Все заметки хранятся в Sidebar → раздел "Заметки". Можно редактировать и удалять.

### Поиск заметок через Command Palette

В Command Palette (`Ctrl+K`) введите `:` (двоеточие) + текст для поиска по заметкам.

## 11. Command Palette

Быстрый доступ ко всем функциям приложения.

### Открыть

`Ctrl+K` (или `Ctrl+K` повторно — закрыть).

### Режимы

| Префикс | Режим |
|---------|-------|
| _(пусто)_ | Команды + последние задачи |
| Текст | Поиск по задачам и командам |
| `>` | Перемещение задачи в другую колонку |
| `:` | Поиск по заметкам |

### Примеры

- `> написать` — показывает задачи с "написать" и колонки куда переместить.
- `: встреча` — ищет заметки со словом "встреча".
- `настройки` — находит команду "Настройки".

### Доступные команды

- Создать задачу
- Быстрая заметка
- AI Помощник
- Настройки
- Сменить тему
- Экспорт / Импорт данных
- Открыть архив
- Виджет вкл/выкл
- Focus Mode

### Навигация

`↑↓` — перемещение, `Enter` — выбрать, `Esc` — закрыть.

## 12. Global Search

Глобальный поиск по всему содержимому приложения — задачам, заметкам и доскам.

### Открыть

`Ctrl+Space` — открыть/закрыть.

### Возможности

- Ищет одновременно по задачам, заметкам и доскам.
- Поддерживает fuzzy-поиск (нечёткое совпадение).
- Подсвечивает найденные фрагменты.
- Показывает к какой доске и колонке относится задача.

### Фильтры

Вверху окна есть вкладки: **Всё / Задачи / Заметки / Доски**. Переключаются кликом или клавишей `Tab`.

### Навигация

- `↑↓` — перемещение по результатам.
- `Enter` — открыть выбранный элемент.
- `Tab` — переключить фильтр.
- `Esc` — закрыть.

### Действия по результату

- Клик/Enter на **задаче** — открывает детальный вид задачи.
- Клик/Enter на **заметке** — открывает заметку в Sidebar.
- Клик/Enter на **доске** — переключает на эту доску.

## 13. Клавиатурная навигация

На доске работает полная навигация без мыши (когда фокус не в поле ввода):

| Клавиша | Действие |
|---------|----------|
| `←` / `→` | Переключить активную колонку |
| `↑` / `↓` | Перемещаться по задачам в колонке |
| `Enter` | Открыть выбранную задачу |
| `N` | Создать новую задачу |
| `/` | Перейти в поиск (Sidebar) |
| `Delete` | Удалить выбранную задачу |
| `1`–`9` | Переместить задачу в колонку №1–9 |

## 14. Мини-виджет

Компактное окно always-on-top, которое всегда поверх других приложений. Показывает задачи из колонки "В работе".

### Открыть/закрыть

- Хоткей `Ctrl+Shift+W`.
- Command Palette → "Виджет вкл/выкл".

### Возможности

- Просмотр задач в работе.
- Клик по задаче — открывает её детальный вид в главном окне.
- Drag & drop файлов/писем прямо на виджет — создаёт задачу.
- Перетаскивание окна за заголовок.

## 15. Настройки

Откройте кнопкой ⚙ в title bar или Command Palette → "Настройки".

### Вкладки

#### Общие

- **Автозапуск** — запускать Task Grabber при старте Windows. Переключатель вкл/выкл.

#### Хоткеи

Кастомизация всех глобальных хоткеев:

| Хоткей | Действие |
|--------|----------|

## 16. AI Помощник

Встроенный чат с AI, который знает ваши задачи. Открывается кнопкой **[AI]** в title bar или через Command Palette.

### Настройка

Перейдите Настройки → вкладка **AI**:

**OpenRouter (облако):**
1. Выберите провайдер "OpenRouter (облако)".
2. Введите API ключ (получить на openrouter.ai).
3. Укажите модель, например `openai/gpt-4o-mini`.
4. Нажмите "Сохранить настройки AI".

**Ollama (локально):**
1. Выберите провайдер "Ollama (локально)".
2. Убедитесь, что Ollama запущен локально.
3. Укажите URL (по умолчанию `http://localhost:11434`).
4. Введите имя модели, например `llama3.2`.
5. Нажмите "Сохранить настройки AI".

### Использование

- Задайте любой вопрос в текстовом поле, нажмите **Enter** или кнопку отправки.
- `Shift+Enter` — перенос строки.
- Кнопка **"Итог дня"** — AI получает список ваших задач и составляет краткий отчёт с рекомендациями.

### Разбивка задач

Попросите AI разбить большую задачу на подзадачи — опишите задачу в чате и задайте вопрос. AI сформирует список шагов с учётом контекста ваших текущих дел.

### Итог дня

Кнопка **"Итог дня"** автоматически:
1. Собирает все активные задачи.
2. Показывает **предпросмотр** — что именно уйдёт на сервер (можно отменить).
3. После подтверждения отправляет задачи в AI и получает краткий отчёт: что сделано, что застряло, топ-3 рекомендации на завтра.

### Конфиденциальность

- Данные задач отправляются на сервер (OpenRouter) или обрабатываются локально (Ollama).
- Задачи, помеченные как **конфиденциальные**, автоматически обфусцируются (заменяются на "CONFIDENTIAL CASE #N").
- Переключатель **"Полностью исключать конфиденциальные"** — такие задачи вообще не попадут в контекст.
- Перед отправкой "Итога дня" показывается **предпросмотр**: можно увидеть, что именно уйдёт на сервер.
- При использовании Ollama данные не покидают ваш компьютер.

## 17. Smart Rules (Умные правила)

Конструктор автоматических правил по принципу **ЕСЛИ → ТО**. Открывается через Настройки → раздел "Умные правила" или Command Palette.

### Принцип работы

Каждое правило: выбираете **условие** (ЕСЛИ) и **действие** (ТО). Правила запускаются автоматически или вручную.

### Создание правила

1. Откройте диалог "Умные правила".
2. Нажмите **+ Новое правило**.
3. Введите название правила.
4. Настройте **ЕСЛИ**:
   - Выберите **поле** (приоритет, колонка, дедлайн, тег, заголовок, источник, дней в колонке, дней без активности).
   - Выберите **оператор** (равно, не равно, содержит, просрочен, больше, меньше, более N дней).
   - Укажите **значение**.
5. Настройте **ТО**:
   - Выберите **действие** (переместить в колонку, установить приоритет, добавить тег, архивировать, установить цвет, уведомить).
   - Укажите **значение** для действия.
6. Нажмите **Сохранить**.

### Примеры правил

| ЕСЛИ | ТО |
|------|-----|
| Приоритет = Высокий | Переместить в "В работе" |
| Дедлайн просрочен | Уведомить "Просрочено!" |
| В колонке более 7 дней | Установить приоритет Высокий |
| Заголовок содержит "баг" | Добавить тег "bug" |
| Без активности более 14 дней | Архивировать |

### Управление правилами

- **Вкл/выкл** — переключатель слева от правила.
- **Изменить** — кнопка редактирования.
- **Удалить** — кнопка корзины.
- **Запустить** — кнопка "Запустить" в заголовке диалога запускает все активные правила прямо сейчас.

## 22. Экспорт, импорт и бэкапы

### Экспорт

Настройки → Данные → **Экспорт JSON**. Откроется диалог сохранения файла. Файл содержит все задачи, колонки, теги, заметки, вложения (пути).

### Импорт

Настройки → Данные → **Импорт JSON**. Выберите ранее экспортированный файл. **Внимание:** текущие данные будут заменены. После импорта требуется перезапуск.

### Бэкапы

Настройки → Данные → раздел **Бэкапы**:

- **Создать бэкап** — мгновенное создание копии базы.
- Хранятся последние **5 бэкапов** с датой и размером.
- **Восстановить** — нажмите кнопку напротив нужного бэкапа (подтверждение → перезапуск).

**Автоматические бэкапы** создаются приложением самостоятельно. Хранятся в папке `AppData\Roaming\task-grabber\backups`.

## 23. Автоматизация

Настройки → **Авто** (вкладка "Авто").

### Автоархивация готовых задач

Задачи в колонках "Готово" и "Забито" старше N дней автоматически уходят в архив. По умолчанию — через **7 дней**.

Настройки: включить/выключить + количество дней.

### Напоминания о просроченных

Если у задачи есть дедлайн и он прошёл, автоматически ставится напоминание. Придёт системное уведомление.

### Залежавшиеся важные задачи

Если задача с высоким приоритетом находится в колонке "Новые" более **3 дней** — придёт уведомление-напоминание.

### Ручной запуск

Кнопка **"Запустить проверку сейчас"** — немедленно выполнить все автоматические правила.

### WIP лимиты

В настройках колонки можно установить **WIP Limit** — максимальное число задач. Если лимит превышен, заголовок колонки подсвечивается предупреждением (жёлтый/красный). Настраивается через редактор колонки (иконка ⚙ рядом с названием колонки).

## Советы по использованию

**Максимально быстрый захват задачи:**
Выделите текст → `Ctrl+Shift+T` дважды быстро → задача создана без диалога, можно продолжать работу.

**Обработка письма из Outlook:**
Перетащите `.msg` файл на доску или виджет → задача создастся автоматически с темой и телом письма.

**Работа без мыши:**
Используйте `Ctrl+K` для всего — поиск задач, навигация, перемещение между колонками, смена темы.

**Поиск по всему:**
`Ctrl+Space` — одно окно для поиска по задачам, заметкам и доскам одновременно.

**Разные контексты:**
Создайте отдельные доски для работы и личных дел. Переключайтесь через BoardSwitcher в title bar.

**Автоматизация:**
Настройте Smart Rules — например, "если задача в колонке более 5 дней → уведомить". Избавит от залежавшихся задач.

**Приватность:**
Пометьте чувствительные задачи как "Конфиденциально" — они будут скрыты от AI в виде CONFIDENTIAL CASE #N.
